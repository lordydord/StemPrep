# Privacy

StemPrep does not include analytics, advertising, crash reporting or a developer-operated backend.

## Data stored on your Mac

- Your MVSEP API token is stored in macOS Keychain.
- App preferences, cached model information, resumable job details and recent-session metadata are stored in your user Library.
- Completed stems, the source copy and the separation manifest are stored in a sibling folder beside the WAV you selected.

## Data sent to MVSEP

When you start a separation, StemPrep sends the selected WAV, your API token, the selected model settings and output format directly to MVSEP. MVSEP processes and temporarily hosts the resulting files. Its handling of that data is governed by the [MVSEP Privacy Policy](https://mvsep.com/en/privacy-policy) and [Terms](https://mvsep.com/en/terms).

StemPrep does not receive or retain a copy of your audio, API token or results outside your Mac and MVSEP.

## Removing local data

- Remove or replace the API token in StemPrep Settings.
- Delete completed stem folders in Finder.
- Remove StemPrep preferences and Application Support data if you want to reset the app completely.

This document applies to StemPrep 1.0.0.
